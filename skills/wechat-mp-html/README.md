# wechat-mp-html（微信公众号图文 HTML）

## 作用

生成符合微信公众号编辑器实践的 **HTML 正文**：`#js_content` 容器、inline 样式约束、标准「复制正文到公众号」脚本等。深度排版规则在同目录 **`references/`**。

排版规范与脚本用法以 **[SKILL.md](SKILL.md)** 为准。

## 安装

```bash
npx skills add baklib-tools/skills --skill wechat-mp-html
```

手动安装：复制 [`skills/wechat-mp-html/`](https://github.com/baklib-tools/skills/tree/main/skills/wechat-mp-html)（含 **`references/`**）。

## 使用

产出渠道稿或 HTML 前确认用户需要该格式，并对照 **`references/`** 与 **[SKILL.md](SKILL.md)** 生成可复制粘贴的 HTML。
