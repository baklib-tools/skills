# baklib-theme-dev（Baklib 站点模板开发）

## 作用

**Baklib 站点模板（主题）**开发：Liquid 目录与命名、对象 / 指令 / 过滤器、静态页、`seeds` / `migrations` 等。**新建主题脚手架**与**参考站复刻工作流**说明在同目录 **`references/`**（如 `create-theme-scaffold.md`、`website-cloning-workflow.md`）。

模板结构与开发流程以 **[SKILL.md](SKILL.md)** 为准。

## 安装

```bash
npx skills add baklib-tools/skills --skill baklib-theme-dev
```

手动安装：复制 [`skills/baklib-theme-dev/`](https://github.com/baklib-tools/skills/tree/main/skills/baklib-theme-dev)（含 **`references/`**）。

## 使用

新建或迁移主题时，先阅读 **`references/`** 中对应篇目，再按 **[SKILL.md](SKILL.md)** 与 Baklib 官方模板约定实施。

## 示例

脚手架与复刻步骤见 **`references/`**；可与本地 Baklib 站点联调验证。
