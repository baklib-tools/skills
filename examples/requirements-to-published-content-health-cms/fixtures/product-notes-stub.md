# 产品要点摘录（无本地知识库时的降级参考）

> **说明**：标准演示应使用团队本地的 **baklib-workspace** 产品知识库根目录（路径请自行指定，见上级 [README](../README.md)）。若你**没有**该目录，可用本页极简要点做「占位」，能力边界仍以 [Baklib 官网](https://www.baklib.com) 与公开文档为准。

- **知识库 / 文档**：在线编辑、版本、协作类能力以当前产品为准。  
- **资源库（DAM）**：媒体上传、标签、合集、引用。  
- **站点 / 渠道**：多站点、模板、页面发布；与「一键发到所有外部平台」通常需**集成或分阶段**。  
- **工作流**：若需**多节点可配置审批链**，需核对是否由产品内置、插件或外部系统集成。  
- **API**：Open API / 集成方式以官方文档为准。

本 stub **不能**替代真实知识库检索；正式售前请使用完整镜像与最新版本说明。
