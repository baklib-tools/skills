# 技能使用示例（Examples）

本目录存放**可复现的示例**：说明如何加载对应技能、如何操作，以及**预期效果**（含截图或产物说明）。示例与仓库根目录 [`skills/`](../skills/) 中的发布技能一一对应或组合使用。

| 示例 | 关联技能 | 说明 |
|------|----------|------|
| [wechat-mp-html-quiz/](wechat-mp-html-quiz/README.md) | [wechat-mp-html](../skills/wechat-mp-html/SKILL.md) | 本地 HTML「小测验」排版，演示 `#js_content` 与一键复制 |
| [requirements-to-published-content-health-cms/](requirements-to-published-content-health-cms/README.md) | [requirements-to-published-content](../skills/requirements-to-published-content/SKILL.md) 等 | 脱敏医疗 CMS 需求 → 分析/方案/软文/公众号 HTML；含生成配图、[公众号 HTML 全页截图](requirements-to-published-content-health-cms/wechat-html-demo-fullpage.png)、[模拟对话](requirements-to-published-content-health-cms/walkthrough/simulated-dialogue.md)；配合本地 `baklib-workspace` |

新增示例时建议：自建子目录、`README.md` 写清步骤与技能名，资源文件（HTML、截图）放在同目录便于引用。
