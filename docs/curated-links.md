# 精选外链

本文件收录与 **Baklib**、本仓库 **Agent Skills** 或常见配套工作流**直接相关**的工具与项目链接；每条附一句用途说明，便于检索与定期回顾。

- **收录原则**：本仓库维护者愿意背书其用途；不追求「全网最全」，与 [external-skills/](../external-skills/) 子模块（整块外部仓库）互补。
- **与技能内文档分工**：仅某一技能适用的长参考请放在对应 `skills/<name>/references/`；此处仅作全库级索引或交叉引用。
